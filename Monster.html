John Heiden's Monster Project

Monster Project

This program was assigned as homework in my intro game development class. Different types of monsters traverse the screen with different behaviors. One monster wraps to the other side of the screen when it hits a wall, one bounces off of walls, one moves in a set pattern, one moves in a random path, and one is controlled by the player. This project shows the diversity of behaviors that a program can create in such similar objects.
